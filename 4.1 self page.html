About Me

안녕하세요! 저는 김민수이며, 사용자에게 가치 있는 서비스를 제공하는 것을 최우선으로 하는 소프트웨어 개발자입니다.

대학에서 컴퓨터공학을 전공하며 프로그래밍의 기초를 다졌고, 졸업 후 2년간 스타트업에서 웹 개발 업무를 담당했습니다. 빠르게 변화하는 IT 업계에서 새로운 기술을 학습하고 적용하는 것에 큰 흥미를 느끼며, 특히 사용자 경험을 개선하는 기술적 솔루션을 찾는 것을 좋아합니다.

저의 생활신조는 "기술은 사람을 위해 존재한다"입니다. 아무리 기술적으로 뛰어난 코드라도 실제 사용자에게 도움이 되지 않는다면 의미가 없다고 생각합니다.

다음은 제가 최근에 작업한 기술들입니다:

  • JavaScript (ES6+)
  • React
  • Node.js
  • TypeScript
  • HTML & CSS
  • Git & GitHub

성격의 강점: 꾸준한 학습 의지, 뛰어난 협업과 소통 능력, 문제 해결 지향적 사고방식
